# HOTS-using-diffusive-memristor
HOTS algorithm using diffusive memristor
Python scripts are modified based on paper "HOTS: A Hierarchy of Event-Based Time-Surfaces for Pattern Recognition, Xavier Lagorce, Garrick Orchard, Francesco Galluppi, Bertram E. Shi, IEEE TRANSACTIONS ON PATTERN ANALYSIS AND MACHINE INTELLIGENCE, VOL. 39, NO. 7, JULY 2017", please check the original script for your research.
